# User Frontend - Movie Rental Dashboard

This React + Vite project implements a user dashboard for the provided Movie Rental backend API.

## Quick Start

```bash
cd user-frontend
npm install
npm run dev
```

## Environment

Set in `.env`:

- `VITE_API_BASE_URL=http://localhost:5000`

## Included Modules

- Auth (register/login/me)
- Movies list & details
- Category filtering
- Payment order + verify scaffolding
- User profile + purchased movies + watch flow
